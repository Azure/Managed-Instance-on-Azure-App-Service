# Managed Instance on Azure App Service

This is a repository to support the public preview of Managed Instance on Azure App Service

This repository contains additional frequently asked questions and example configuration/installation scripts based on common customer scenarios.  Any script provided in this repository should be used as an example only.  You are free to use the scripts but must modify and validate for your own scenario and use.

The documentation for Managed Instance on Azure App Service can be found on [learn.microsoft.com](https://aka.ms/managedinstanceonappservicedocs) within the Azure App Service product documentation.
